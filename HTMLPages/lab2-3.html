Software Development Life Cycle

  1. Define the problem

    1. This step involves gathering the requirements of the program needed to solve the problem.
    2. Questions that should be answered:
      • What data do we need to process?
      • How should we process the data?
      • What output should the program deliver?
      • How should the user interface look?

  2. Design the Program

    1. Programns design focuses on breaking down the program into smaller, more manageable componets. various tools are used to design a program.
    2. Commonly used tools:
      • Structure charts
      • algorithm
      • flow chart
      • decision table
      • pseduocode

  3. Code the Program

    1. Once the program has been designed, a developer then writes the code.
    2. Coding often invloves thanslation the design from charts and pseduocode into a specific programing language. Usually the language is choosen during the "defining" stage.

  4. Testing and Debugging

    1. after code has been written the stage resolves around fixing any bugs in the program.
    2. There are two kinds of errors that cause bugs:
      • Syntax errors: These are grammatical mistakes in the code.
      • Logical error: These mistakes might cause an other than intended output but the code still works.
    3. The programmer must fix these mistakes by carefully examaning the already written code for errors.

  5. Documentation

    1. After the code has been completed, the programmer must write a manual that generally contains the following
      • An overview of the programs functionality
      • In-depth explanations of the program features
      • documentaition of all commands
      • A thorough description of error meddages generated by the program

  6. Deploying and Maintaining the Program

    1. A software team will be responsible for fixing new errors and updating the softwawre with new features.
    2. when a new feature should be added to the program, the cycle restarts as developers begin to define what the new feature should solve.
